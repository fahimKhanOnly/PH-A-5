Important FAQ

What is the Document Object Model (DOM)?

DOM is a programming interface for web documents. It represents the structure of a webpage as a hierarchical tree of objects.

How do you select an element from the DOM?

We can select Element using these methods getElementById(), getElementsByTagName(), querySelector(), querySelectorAll(), etc.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a way of propagation to manage events efficiently. It allows to adding event listener in a single-parent element. This event listener takes advantage of event bubbling to handle its children.

How do you manipulate an element's attributes and styles using the DOM?

We can Manipulate attributes using these methods setAttribute(), getAttribute(), removeAttribute(), etc. and We can manipulate styles using this property "style". Also using these methods in classList.add(), classList.remove(), etc.
